# Python Basics Practice

## Description
This project contains exercises to practice Python basics including:
- Print statements and arithmetic
- String manipulation
- User input
- PCEP exam practice questions

## Instructions
1. Run each script in a Python environment or command prompt:
   - `part1_basic_operations.py`
   - `part2_string_exercises.py`
   - `part3_user_input.py`
2. Review `lab1_answers.txt` for Part 4 answers.

## Requirements
- Python 3.x
